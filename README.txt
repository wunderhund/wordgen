-----------
Some Notes:
-----------
The wordgen script requires python 3.x to run; ideally 3.3 or later.

Python 3.4 can be freely downloaded at:
http://www.python.org/download/

The latest version of Python3 can be installed on Debian-based linux distributions using:
apt-get update
apt-get install python3

Some Windows users may prefer Activestate Python available at:
http://www.activestate.com/activepython/downloads
